# Human Pack v1 Concept Gate

`family-gate-v1.png` is the first original five-master family-language concept
for Human Pack v1. It was generated from the owner-approved art brief with the
Human Mascot Reference used only as a quality and coverage reference.

Left to right:

1. H01 — Black girl, child, coily two-puff hairstyle;
2. H04 — Black young adult man with an articulated lower-limb prosthesis;
3. H07 — Southeast Asian adult man using a manual wheelchair;
4. H12 — East Asian middle-aged woman with a behind-the-ear hearing aid;
5. H13 — Black senior woman using a rollator.

The project owner approved this image on 2026-07-15 as the canonical family
foundation for the Human Character Library. It establishes the visual language,
anatomical rules, assistive-device integration, age diversity, and identity
principles future human characters must follow. It is deliberately the
foundation, not the complete library.

This raster image remains a reference, not a production pack asset or layered
rig source. Redistribution remains blocked until the project owner records the
intended asset license. Production work requires original layered SVG masters,
turnarounds, device behavior sheets, small-size checks, and the complete review
gate in `docs/HUMAN_PACK_VISUAL_STANDARD.md`.
