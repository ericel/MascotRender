# Sticker display font set v1

MascotRender bundles these static TTF files for deterministic, pack-local
caption rendering. Each family is locked to an authoritative upstream revision,
has a complete local SIL Open Font License 1.1, and is identified by SHA-256 in
`manifest.json`.

The set provides four deliberately different chat voices:

- Changa One — compact punch;
- Bangers — energetic comic/slanted display;
- Lilita One — soft rounded display;
- Kalam Bold — handwritten emphasis.

Fonts.Free was used for discovery only. Redistribution is based on the font
authors' files in the official `google/fonts` repository, not a third-party
download mirror.
